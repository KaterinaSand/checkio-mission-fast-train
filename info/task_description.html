You are planning the service of the train. You want to know the minimum time between stations.

Each section of the rail between stations is given in the list. Each section is a tuple of distance and speed limit (Both are integers). You can change speed ( +1. -1 and ± 0 ) at start and every minute. The train runs by the same amount as the speed value.

  • Starting speed is 0.
  • Speed limit is set for each section of the rail. You don't exceed this.
  • You must reach the target station at speed 1.(Because it is necessary to stop at the station)

You should return the minimum time (minutes) as an integer.

Example:

fast_train([(4, 3)]) == 3 # (1, 2, 1)
fast_train([(9, 5)]) == 5 # (1, 2, 3, 2, 1)
fast_train([(5, 5), (4, 2)]) == 6 # e.g. (1, 2, 2, 2, 1, 1)

Input: List of the section of the rail. Each section is tuple of distance (as an integer) and speed limit (as an integer).

Output: The minimum time (minutes) as an integer.

Precondition:

  • all(distance) > 0
  • all(speed > 0)
  • len(section) > 0
